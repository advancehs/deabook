# 10. 随机非参数数据包络分析

在效率分析领域，虽然尚未形成统一的理论框架，但数据包络分析（DEA）和随机前沿分析（SFA）已发展成为两种具有互补特征的主流方法。这两种方法论在处理效率评估问题时呈现出不同的技术路径。方法论的选择往往取决于研究数据的特征和具体问题的需求。

数据包络分析（DEA，Farrell 1957；Charnes et al. 1978）[^1] [^2]是一种公理的、数学编程的效率分析方法。与计量经济学和基于回归的工具相比，DEA的主要优势在于它对前沿的非参数化处理，建立在生产理论的公理之上，如自由处置性（单调性）、凸性（凹性）和规模收益不变（同质性）。DEA并不假定边界或无效率分布的任何特定函数形式。它的直接的、以数据为导向的方法有助于将效率分析的结果传达给决策者。然而，DEA的主要缺点是它把所有偏离前沿的情况都归结为无效率。这往往是一个非常强的假设。

SFA的主要优势在于其对偏离前沿的概率建模，这些偏离被分解为一个非负的无效率项和一个特异性误差项，该误差项考虑了一些遗漏的因素，如企业及其经营环境的非观察异质性、测量和数据处理的随机误差、规格误差和其他噪音来源。与DEA相比，SFA采用了参数化的回归技术，这需要事先对前沿的函数形式和无效率分布进行规范。由于经济理论很少证明一个特定的函数形式是合理的，所以经常使用灵活的函数形式，如translog。然而，灵活的函数形式往往违反了生产理论的公理，而强加这些公理则会降低灵活性。

鉴于不同方法论之间的界限与区别，凸非参数最小二乘法（Convex Nonparametric Least Squares, CNLS）的提出（Kuosmanen，2008）[^3]开创了新的研究路径。CNLS通过二次规划方法估计满足单调性与凸性约束的非参数前沿，既继承了DEA的公理化建模优势，又融合了SFA的统计推断框架。基于CNLS的拓展研究（Kuosmanen & Johnson 2010；Kuosmanen & Kortelainen 2012）[^4] [^5] 最终形成了统一的分析框架——随机非参数数据包络（Stochastic Nonparametric Envelopment of Data, StoNED）。该框架实现了生产技术的非参数估计与误差成分的统计分解，为效率分析提供了更为严谨的计量经济学基础。

## 10.1 单产出模型设定

首先，我们先研究多投入、单产出的情形，与SFA模型进行比较，以便理解二者的差异。多投入、多产出的情形以及多投入、多产出且有非期望产出的情形是单产出模型的扩展，我们在本节后接着讨论。

首先，假设生产技术由前沿生产函数$f(\bold{x})$表示，其中$\bold{x}$是一个m维的投入向量。前沿$f(\bold{x})$表示用投入$\bold{x}$可以生产的最大产出，因此，函数$f(\bold{x})$表征了生产可能性集的边界。我们假设函数$f(\bold{x})$属于连续的、单调递增的、全局凹的函数类，可以是不可微的（我们把这个类表示为$F_2$。这相当于说明生产可能性集满足自由可处置性和凸性的经典DEA假设。与SFA不同的是，没有假定$f(\bold{x})$的具体函数形式。

由于效率低下和噪声的影响，DMU$_i$ 的被观察到的产出 $y_i$ 可能与 $f(\bold{x_i})$ 不同。我们遵循SFA文献，引入一个综合误差项 $\varepsilon_{i}=v_i-u_i$，它由非效率项$u_i$ 和随机噪声项$v_i$组成，形式为
$$
y_{i}=f(\bold{x_i}) +{\varepsilon_{i}}\\
 =f(\bold{x_i}) +v_i-u_i, \quad i=1, \ldots, n \tag{10.1}
$$

式中，变量 $u_i$ 和 $v_i$ 是随机变量，假定 $u_i$ 和 $v_i$ 相互独立，且 $u_i$ 和 $v_i$ 都与投入 $\bold{x}$ 无关。另外，假定 $ E\left(u_{i}\right)=\mu>0$，$Var\left(u_{i}\right)=\sigma^2_u<\infty$，且$ E\left(v_{i}\right)=0 $，$Var\left(u_{i}\right)=\sigma^2_v<\infty$。假设$\sigma^2_u$和$\sigma^2_v$在不同DMU间是恒定的，这被称为同方差性；具有异方差性的非效率和噪声模型需要对$u_i$和$v_i$的分布进一步施加假设，这里不做讨论，可见 Kuosmanen (2008) [^3] 的进一步讨论。在这个模型中，我们没有对 $f(\bold{x_i})$ 的具体函数形式施加限制，这与DEA文献的方法相同。另外，我们假设综合误差项 $\varepsilon_{i}$ 是由非效率项 $u_i$ 和随机噪声项$v_i$组成的，这与SFA文献的方法相同。因此，这个模型结合了DEA和SFA方法的优点。然而，与DEA模型相比，这个模型对非效率项 $u_i$ 的分布施加了假设。另外，与SFA模型相比，对 $u_i$ 和 $v_i$ 的分布的假设可能无法进一步精细化。

除了假设综合误差项 $\varepsilon_{i}=v_i-u_i$ 外，我们可以施加一个替代性的假设：$\varepsilon_{i}=-u_i$，从而与DEA模型的假设保持相似。需要注意的是，这里的非效率项$u_i$不是最终需要的非效率，非效率的计算方法为$u_i$除以前沿（$y_i+u_i$）。另外，我们可以施加另外的替代性假设：$\varepsilon_{i}=v_i$，从而与凸非参数最小二乘法的假设相同。下面，我们一一介绍这些模型。


### 10.1.1 凸非参数最小二乘法

当我们把模型（10.1）中综合误差项 $\varepsilon_{i}=v_i-u_i$ 的假设替换为综合误差项 $\varepsilon_{i}=v_i$ 时，我们便可得到凸非参数最小二乘的模型，即：
$$
y_{i}=f(\bold{x_i}) +{\varepsilon_{i}}\
 =f(\bold{x_i}) +v_i, \quad i=1, \ldots, n
$$
其中，变量$v_i$是随机变量，假定$v_i$与投入$\bold{x}$无关， $ E\left(v_{i}\right)=0 $，$Var\left(u_{i}\right)=\sigma^2_v<\infty$。假设$\sigma^2_v$在不同DMU间是恒定的。此时，$E(y_{i}|X)=f(\bold{x_i}) +E(v_i|X)=f(\bold{x_i}) +0$。在满足这种假设的条件下，凸非参数最小二乘法估计量是以下广义回归问题的解：
$$
\min _{f \in F_2}  \sum_{i=1}^{n}\left(y_{i}-f\left(\mathbf{x}_{i}\right)\right)^{2}, \tag{10.2}
$$
其中，$F_2$ 代表与投入$\bold{x}$维度相同的的凹函数空间。模型（10.2）是一个关于函数的优化问题，这个问题可以等效地写成一个有限维的凹二次优化问题，其中变量是函数值和每个点$\bold{x_1},...,\bold{x_n}$的子梯度(Boyd和Vandenberghe, 2004; Kuosmanen, 2008) [^6] [^3]。作为约束条件的一部分，我们强制执行凹性条件；也就是说，凹函数的图形位于其每个切线平面之上。更确切地说，这个凹性条件意味着，给定任何一点$\bold{x_i}$，$f$在$\bold{x_i}$处的值大于或等于$f$在$\bold{x_i}$处估计的任何切线超平面的值。显然，任何凹函数在每一点都有一个非空的子梯度，而且这种切线平面的存在是有保证的。对于这个问题，在有限样本下，对所有 $n(n-1)$ 对观测点 $\bold{x}_i, \bold{x}_j \, (1 \leq i,j \leq n, \, i \neq j)$ 施加凹性约束条件：$
f(\bold{x}_j) \leq f(\bold{x}_i) + \nabla f(\bold{x}_i)^\top (\bold{x}_j - \bold{x}_i)$，即可保证函数 $f$ 在整个定义域上的全局凹性。这是因为凹函数在任意两点间的线性插值均不高于其实际值，而样本外点的性质可通过插值/外推自动满足（无需显式约束）。

由此形成带有变量${\alpha}, \boldsymbol{\beta}, {\varepsilon}$的二次优化问题如下：
$$
\begin{array}{l}
\min _{{\alpha}, \boldsymbol{\beta}, {\varepsilon}} \sum_{i=1}^{n}\left({\varepsilon_i^{CNLS}}\right)^{2} \\
\text { S.t. } \\
{y}_{i}=\alpha_{i}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{i}+{\varepsilon_i^{CNLS}} &\forall i=1, \ldots, n \\
\alpha_{i}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{i} \leq \alpha_{h}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{h} &\forall h, i=1, \ldots, n \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i=1, \ldots, n
\end{array}       \tag{10.3}
$$

其中，变量$\alpha_{i}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{i}$代表$f(\bold{x_i})$的切线，$\boldsymbol{\beta}_{i}$属于凸函数$f$在每个$\bold{x_i}$的子梯度，也就是边际产量。这个问题的解$\hat\alpha+\mathbf{x}' \hat {\boldsymbol{\beta}}$被称为凸非参数最小二乘法估计量(OLS)。注意，我们通过设置$\alpha_i=\alpha,\boldsymbol{\beta}_{i}=\boldsymbol{\beta} \forall i$和$f(\bold{x_i})=\alpha+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}$来恢复普通最小二乘线性回归问题。也就是说，CNLS估计量估计了n条不同的回归线，其得到的截距和斜率系数因DMU而异，而不是像OLS那样只拟合一条回归线。模型的第一个约束施加了产出观测值与超平面之间的关系。第二个约束通过应用 Afriat 不等式系统来强加凹性。 第三个约束强加了单调性。

可以证明，在满足上述假设时，$\hat f^{CNLS}(\bold{x_i})$是$f(\bold{x_i})$的无偏和一致估计。特别是，$ E\left(v_{i}\right)=0 $的假设对一致性的估计至关重要，当违反这个假设时，CNLS估计量将不是一致的估计。

另外，$\hat f^{CNLS}(\bold{x_i})$在样本内观察值$(\bold{x_i},y_i)$上是$f(\bold{x_i})$的唯一估计。虽然CNLS估计量对于样本内观察值是唯一的，但是没有唯一的方法将CNLS估计量外推到样本外观察值。CNLS外推的不唯一性导致样本外预测结果波动（类似过拟合中高方差模型对噪声敏感），因模型对样本内结构过度适配而缺乏稳健性。为了使CNLS估计量满足样本外的观察值,Bertsimas和Mundru (2021)[^7]建议使用正则化，即加入$\boldsymbol{\beta}_{i}$的二范数，得到的需要求解的问题如下：
$$
\begin{array}{l}
\min _{{\alpha}, \boldsymbol{\beta}, {\varepsilon}} \sum_{i=1}^{n}\left({\varepsilon_i^{CNLS}}\right)^{2} +{\lambda} \sum_{i=1}^{n}\left\|\xi_{i}\right\|^{2} \\
\text { S.t. } \\
{y}_{i}=\alpha_{i}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{i}+{\varepsilon_i^{CNLS}} &\forall i=1, \ldots, n \\
\alpha_{i}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{i} \leq \alpha_{h}+\mathbf{x}_{\boldsymbol{i}}' \boldsymbol{\beta}_{h} &\forall h, i=1, \ldots, n \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i=1, \ldots, n
\end{array}       \tag{10.4}
$$

另一方面，Kuosmanen和Kortelainen（2012）[^5]呼吁使用最小外推原则，并建议使用下限值：
$$
\hat f_{min}^{CNLS}(\bold{x})=min_{\alpha,\bold{\beta}} \{\alpha+\bold{x'}|\alpha_i+\bold{x_i'}\geq\hat f^{CNLS}(\bold{x_i}) \forall i=1,...,n\} \tag{10.5}
$$

### 10.1.2 确定性前沿

在这一节中，我们考虑模型（10.1）的另一种特殊情况，即综合误差项$\varepsilon_{i}=v_i-u_i$的假设替换为综合误差项$\varepsilon_{i}=-u_i$时，我们便可得到确定性模型。确定性模型中的模型规格和投入和产出数据被假定为准确无误的，所以唯一的概率成分是从生产可能性集中抽取的随机观察样本。因此，DEA 的 bootstrap 置信区间不能直接与其他真正具有随机性的模型的信心区间（如 SFA 置信区间）进行比较。因此，如果确定性模型的无噪声假设$(v_i=0)$不成立，DEA的统计基础就会崩溃。调整小样本的bootstrapping方法并不是针对噪声的补救措施。

在单一产出的情况下，第$o$个DMU的生产函数$f_i$的可变规模收益（VRS）DEA估计值可以被表述为：
$$
\max _{\theta_{o}, \lambda}\left\{\theta_{o} \mid \begin{array}{ll}
\sum_{i=1}^{n} \lambda_{i} y_{i} \geq \theta_{o}y_{o} & \\
\sum_{i=1}^{n}\lambda_{i} x_{i m} \leq x_{o m} & \forall m=1, \ldots, M \\
\sum_{i=1}^{n}\lambda_{i}=1 & \\
\lambda_{i}\geq 0 & \forall i=1, \ldots, n
\end{array}\right\}   \tag{10.6}
$$
乘法模式可以转换为加法模式：


$$
\max _{\phi_{o}, \lambda, \mu}\left\{\varnothing_{o} \mid \begin{array}{ll}
\sum_{i=1}^{n} \lambda_{i} y_{i} \geq y_{o}+\varnothing_{o} & \\
\sum_{i=1}^{n}\lambda_{i}x_{i m} \leq x_{o m} & \forall m=1, \ldots, M \\
\sum_{i=1}^{n}\lambda_{i}=1 & \\
\lambda_{i} \geq 0 & \forall i=1, \ldots, n
\end{array}\right\}
$$

其中 $ \theta_{o}=1+\frac{\varnothing_{o}}{y_{0}} $。

上述线性规划问题的对偶为：
$$
\min _{\alpha, \bold{\beta}}\left\{\alpha+\bold{x'_o\beta}-y_o \mid \begin{array}{ll}
\alpha+\bold{x'_i\beta} \geq y_i  &  \forall i=1,\ldots,n \\
\bold{\beta} \geq0
\end{array}\right\}
$$

引入一个新变量${\varepsilon_o}=y_o-(\alpha+\bold{x'_o\beta})$，并添加这个新约束，上述线性规划问题可以被写为：

$$
\min _{\alpha, \bold{\beta} ,\varepsilon}\left\{-\varepsilon_o \mid \begin{array}{ll}
{\varepsilon_o}=y_o-(\alpha+\bold{x'_o\beta}) \\
\alpha+\bold{x'_i\beta} \geq y_i  &  \forall i=1,\ldots,n \\
\bold{\beta} \geq0
\end{array}\right\}
$$
DEA问题是对每一个DMUo分别求解上述线性规划问题，我们使用新的线性规划问题同时求解所有DMU的线性规划问题：
$$
\min _{\alpha_i, \bold{\beta_i} ,\varepsilon_i}\left\{-\sum_{i=1}^{n} \varepsilon_i \mid \begin{array}{ll}
{\varepsilon_i}=y_i-(\alpha_i+\bold{x'_i\beta_i}) &  \forall i=1,\ldots,n  \\
\alpha_h+\bold{x'_i\beta_h} \geq y_i  &  \forall i,h=1,\ldots,n \\
\bold{\beta_i} \geq0 &  \forall i=1,\ldots,n
\end{array}\right\}
$$

通过构造，可知$\varepsilon_i \leq 0$。另外，由于单调性，上述线性规划问题为：

$$
\min _{\alpha_i, \bold{\beta_i} ,\varepsilon_i}\left\{-\sum_{i=1}^{n} \varepsilon_i \mid 
\begin{array}{ll}
{\varepsilon_i}=y_i-(\alpha_i+\bold{x'_i\beta_i}) &  \forall i=1,\ldots,n  \\
\alpha_h+\bold{x'_i\beta_h} \geq \alpha_i+\bold{x'_i\beta_i}   &  \forall i,h=1,\ldots,n \\
\bold{\beta_i} \geq0,  \varepsilon_i\leq0 &  \forall i=1,\ldots,n
\end{array}\right\}  \tag{10.7}
$$
这是模型（10.3）的限制符号的版本。因此，我们得到的结论是：限制符号版本的CNLS估计量等同于DEA VRS估计量。另外，可以将其他规模收益假定的DEA估计量与限制符号版本的CNLS估计量等价。如若假定DEA的规模报酬不变（CRS），则可以设定${\alpha_i} =0$。若假定DEA的规模报酬非递增（NIRS），则可以设定${\alpha_i} \geq0$。若假定DEA的规模报酬非递减（NDRS），则可以设定${\alpha_i} \leq 0$。另外，这种方法与Aigner 和 Chu (1968)[^8]的参数化规划 (PP) 模型的非参数化扩展类似。

在参数模型中，除了限制综合误差项的符号，还可以通过修正的OLS（COLS）来估计非效率。COLS 是一个两阶段的过程：在第一阶段，前沿是通过普通最小二乘法 (OLS) 回归估计的；在第二阶段，前沿向上移动，使得产生的 COLS 前沿包含所有数据。请注意，普通最小二乘问题 (OLS) 对误差项没有限制，并且 OLS 残差既有正值也有负值。误差的OLS估计量的将表示为$\varepsilon_i^{OLS}$ 。 在 COLS 模型中，这些误差估计量被归因于效率低下； 因此误差的估计量被调整为
$$
\begin{array}{ll}
{\hat\varepsilon_i^{COLS}}={\varepsilon_i^{OLS}}- \underset{i}{max}{\varepsilon_i^{OLS}} &  \forall i=1,\ldots,n 
\end{array}  \tag{10.8}
$$
其中，${\hat\varepsilon_i^{COLS}}$的值范围为$[-\infty,0]$ ，0 代表最优前沿，即最高效。 类似地，我们将截距项调整为
$$
\begin{array}{ll}
{\hat\alpha_i^{COLS}}={\alpha_i^{OLS}} + \underset{i}{max}{\alpha_i^{OLS}} &  \forall i=1,\ldots,n 
\end{array}  \tag{10.9}
$$
斜率系数${\hat\beta_i^{COLS}}$直接从OLS估计中获得，因为${\hat\beta_i^{COLS}}={\hat\beta_i^{OLS}}$。

因CNLS是OLS的非参数扩展，我们可以将COLS中的第一步OLS估计替换为CNLS估计，并同样将COLS的第二步估计非效率。Kuosmanen and Johnson (2010) [^4] 将这种方法称为$C^2 NLS$，完整的估计步骤如下：

**第一步**：通过解决 CNLS 估计量（式10.3）来估计$E(y_i|\bold{x_i)}$，将得到的残差记为${\hat\varepsilon_i^{CNLS}}$。 注意，这里的$E(u_{i}|\bold{x})=-\mu \neq 0$，所以，这里估计的是$h=f+E(u_i)$。

**第二步**：类似于 COLS 过程移动残差，$C^2NLS$效率估计量是
$$
\begin{array}{ll}
{\hat\varepsilon_i^{C^2NLS}}={\varepsilon_i^{CNLS}}- \underset{i}{max}{\varepsilon_i^{CNLS}} &  \forall i=1,\ldots,n 
\end{array}  \tag{10.10}
$$
其中，${\hat\varepsilon_i^{C^2NLS}}$的值范围为$[-\infty,0]$， 0 表示最优前沿，即最高效。 虽然我们已经说明CNLS 估计器在观察点$\bold{x_i},\forall i=1,\ldots,n$上是唯一的，但在未观察的$\bold{x}$ 上不是。因此，可以使用公式（10.5）中的方法来得到$h=f+E(u_i)$的下限$_{min}^{CNLS}(\bold{x})$：
$$
\hat h_{min}^{CNLS}(\bold{x})=min_{\alpha,\bold{\beta}} \{\alpha+\bold{x'\beta}|\alpha_i+\bold{x_i'\beta}\geq\hat h^{CNLS}(\bold{x_i}) \forall i=1,...,n\}
$$
类似地，我们将$C^2NLS$的截距项调整为
$$
\begin{array}{ll}
{\hat\alpha_i^{C^2NLS}}={\alpha_i^{CNLS}} + \underset{i}{max}{\alpha_i^{CNLS}} &  \forall i=1,\ldots,n 
\end{array}  \tag{10.11}
$$
 $C^2NLS$的斜率系数${\hat\beta_i^{C^2NLS}}$直接从CNLS估计中获得，因为${\hat\beta_i^{C^2NLS}}={\hat\beta_i^{CNLS}}$。 $C^2NLS$估计的前沿为：
$$
\begin{array}{ll}
{\hat f^{C^2NLS}}={\hat\alpha_i^{C^2NLS}}+{\hat\beta_i^{C^2NLS}} + \underset{i}{max}{\alpha_i^{CNLS}} &  \forall i=1,\ldots,n 
\end{array}  \tag{10.12}
$$
Kuosmanen和Johnson (2010)[^4]证明了 $C^2NLS$估计量的一致性，即：$\underset{n \to \infty}{lim}{\hat\varepsilon_i^{C^2NLS}}={\varepsilon_i}   \forall i=1,\ldots,n$。值得说明的是，$C^2NLS$的一致性要求所有 DMU 的非效率项都来自相同的分布（同方差）。如果违反 i.i.d.假设（例如，不同DMU的非效率具有不同分布），则无法保证$C^2NLS$的一致性。

### 10.1.3 随机非参数(StoNEd)模型

现在，我们可以使用模型（10.1）中综合误差项$\varepsilon_{i}=v_i-u_i$来进行分析，这样的模型是随机非参数模型，Kuosmanen and Kortelainen (2012)[^5]提出一种StoNEd方法来估计这个问题。可以发现，$E(y_i|\bold{x})=f(\bold{x})+E(\varepsilon_{i}|\bold{x})=f(\bold{x})+(E(v_{i}|\bold{x})-E(u_i)|\bold{x})=f(\bold{x})+(-\mu)$，其中$E(\varepsilon_{i}|\bold{x})=-\mu$。所以，对前沿$f$的估计将不是一致的。所以，可以将综合误差项的条件期望中的非0项与前沿$f$合并估计，这样得到的新的误差项的条件期望就为0，即估计：
$$
\begin{array}{ll}
y_{i}=f(\bold{x_i}) +v_i-u_i,  \\
=(f(\bold{x_i}) -\mu)+ (\mu+v_i-u_i)  \\
=g(\bold{x_i})+ (\mu+v_i-u_i)
\quad i=1, \ldots, n
\end{array}  \tag{10.13}
$$
此时，$E(y_i|\bold{x})=g(\bold{x})+E(u+v_i-u_i|\bold{x})=g(\bold{x})+0$，其中$E(\mu+v_i-u_i|\bold{x})=0$。注意，这里分析的是同方差的情况，如果是异方差的情况，还需要对综合误差项的组成部分的分布的假设。

在同方差的条件下，我们可以将StoNED方法的第一步设定为：可以使用CNLS估计量对模型（10.13）进行估计，得到$g(\bold{x})=f(\bold{x})-\mu$的一致估计。估计的方差和公式（10.3）一致，但是需要注意的是，我们这里使用CNLS估计量估计的残差为对$\mu+v_i-u_i$的估计。

进一步，只需得到$\mu$，就可以得到前沿$f$的一致估计。因此，第二步，我们使用一些不同方法来估计非效率的期望$\mu$。下面介绍使用Fan等人 (1996) [^8]提出的准似然估计和 Hall和Simar（2002）[^9]提出的非参数核密度估计法来估计非效率的非条件期望$\mu$。

#### （一）准似然估计

准似然估计法需要一些额外的参数化分布假设。在下文中，我们将讨论通常假设的半正态非效率和正态噪声的情况。从形式上讲，我们假设：
$$
u_{i} \sim N^{+}\left(0, \sigma_{u}^{2}\right)\\
v_{i} \sim N\left(0, \sigma_{v}^{2}\right)  \tag{10.14}
$$
已知信噪比$\lambda$的计算方法为$ \lambda=\sigma_{u} / \sigma_{v} $，而Fan等人(1996)[^8]证明准似然函数可以被表述为信噪比$\lambda$的函数，即：
$$
\ln L(\lambda)=-n \ln \hat{\sigma}+\sum_{i=1}^{n} \ln \Phi\left[\frac{-\hat{\varepsilon}_{i} \lambda}{\hat{\sigma}}\right]-\frac{1}{2 \hat{\sigma}^{2}} \sum_{i=1}^{n} \hat{\varepsilon}_{i}^{2}  \tag{10.15}
$$
其中，
$$
\begin{array}{c}
\hat{\varepsilon}_{i}=\hat{\varepsilon}_{i}^{C N L S}-(\sqrt{2} \lambda \hat{\sigma}) /\left[\pi\left(1+\lambda^{2}\right)\right]^{1 / 2} \\
\hat{\sigma}=\left\{\frac{1}{n} \sum_{j=1}^{n}\left(\hat{\varepsilon}_{i}^{C N L S}\right)^{2} /\left[1-\frac{2 \lambda^{2}}{\pi(1+\lambda)}\right]\right\}^{1 / 2} .
\end{array}   \tag{10.16}
$$
符号 $ \Phi $ 表示标准正态分布$ \mathrm{N}(0,1) $的累积分布函数。因此，通过最大化准似然估计可以得到$\lambda$的值。又因为${\sigma}^2= \sigma_{v}^{2}+ \sigma_{u}^{2}$，所以：
$$
\hat{\sigma}_{v}=\sqrt{\frac{\hat{\sigma}^2}{1+\lambda^2}} \\
\hat{\sigma}_{u}=\lambda\hat{\sigma_{v}}  \tag{10.17}
$$
在得到$\hat{\sigma_{v}^{2}}$和$\hat{\sigma_{u}^{2}}$之后，可以通过下式计算$u$的无条件均值$\hat{\mu}$：
$$
\hat{\mu}=\sqrt{\frac{2\hat{\sigma}^2\lambda^2}{\pi(1+\lambda^2)}}  \tag{10.18}
$$

#### （二）非参数核密度估计法

与准似然估计不同，非参数核密度估计法的策略是寻找残差密度中的不连续点。逻辑是，如果非效率项被截断，为了代表有效的绩效，分布中一定有一个不连续的地方。当非效率被噪声卷积时，其特征是一个连续而平滑的函数，不连续性仍然存在于卷积变量的密度中，即估计的残差密度。因此，Hall和Simar[^9]建议使用核方法估计残差密度，并使用这些估计值来确定分布右边的导数的最大变化。然后在同方差噪声和非效率的假设下，导数的最大变化的位置可以用来估计样本中的平均非效率。这种方法不依赖于残差的偏度或特定的分布假设。

用$f_{\varepsilon^+}$表示$\varepsilon^+=u+v_i-u_i=u+\varepsilon$的核密度估计器。Hall和Simar(2002)[^9]的研究表明，综合误差项的密度函数的一阶导数$f_\varepsilon'$与$μ$附近的非效率项的一阶导数$f_u'$成正比。这是由于假设$f_u$在零处有一个跳跃不连续。因此，非效率的期望$μ$的稳健非参数估计值为
$$
\hat{\mu}=\arg \max _{z \in \Im}\left(\hat{f}_{\varepsilon^{+}}^{\prime}(z)\right), \tag{10.19}
$$
其中，$ \mathfrak{I} $ 是 $ f_{\varepsilon^{+}} $右尾的一个封闭区间。

接下来，我们便可以使用第一步得到的$g$的CNLS估计量和第二步得到的非效率的期望来计算前沿$f$。但是，我们已经说明，CNLS 估计器在观察点$\bold{x_i},\forall i=1,\ldots,n$上是唯一的，但在未观察的$\bold{x}$ 上不是。因此，Kuosmanen和Kortelainen（2012）[^5]建议应用$g$的下限（类似于公式（10.5）），定义为
$$
\hat g_{min}^{CNLS}(\bold{x})=min_{\alpha,\bold{\beta}} \{\alpha+\bold{x'\beta}|\alpha_i+\bold{x_i'\beta}\geq\hat g^{CNLS}(\bold{x_i}) \forall i=1,...,n\} \tag{10.20}
$$

随后，我们可以提出StoNED方法的第三步：在$g$的CNLS估计量中加入非效率的期望$u$来估计前沿$f$，使用：
$$
\hat f^{Stoned}(\bold{x})=\hat g_{min}^{CNLS}(\bold{x})+\hat u
$$
我们必须强调，即使第三步在统计上使用无偏和一致的方法来估计前沿的$f$，但是企业特定非效率$u_i$并没有被估计出。在正态-半正态情况下，Jondrow等人（1982）（JLMS）[^10]开发了一个给定 $ \varepsilon_{i}      $的非效率 $u_{i}$ 的条件分布公式。常用的JLMS非效率估计值是条件平均数 $ E\left(u_{i} \mid \varepsilon_{i}\right) $。给定参数估计值 $ \hat{\sigma}_{u} $ 和$ \hat{\sigma}_{v} $，非效率的条件期望值可以计算为：
$$
E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)=\frac{\hat{\sigma}_{u} \hat{\sigma}_{v}}{\sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\left[\frac{\phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}{1-\Phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}-\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right]   \tag{10.21}
$$
其中，$ \phi $是标准正态分布$ N(0,1)$ 的密度函数，$\Phi$是相应的累积分布函数，且
$$
\hat{\varepsilon}_{i}=\hat{\varepsilon}_{i}^{C N L S}-\hat{\sigma}_{u} \sqrt{2 / \pi}
$$
得到非效率的条件期望值$E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)$后，可以使用产出的真实值$y_i$与产出的条件前沿值之间的比值得到条件效率：
$$
TE=\frac{y_i}{y_i+E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)} \tag{10.22}
$$


Kuosmanen和Kortelainen(2012) [^5]提出，StoNED模型也可以通过对不限制符号的CNLS中改变${\alpha_i}$的限制条件来获得。如若假定DEA的规模报酬不变（CRS），则可以设定${\alpha_i} =0$。若假定DEA的规模报酬非递增（NIRS），则可以设定${\alpha_i} \geq0$。若假定DEA的规模报酬非递减（NDRS），则可以设定${\alpha_i} \leq 0$。

### 10.1.4 乘法模型

虽然 CNLS 回归很容易适应替代 RTS 假设，但对效率估计的影响有些棘手。具体而言，如果估计受 CRS 约束的平均实践技术 g，然后将边界向上移动 大小为估计的预期无效率，则所得最佳实践边界通常不满足 CRS。这是由于 (8.1) 中假设的低效率和噪声项的加性结构与规模属性的乘法性质不匹配。如果强加 CRS、NIRS 或 NDRS 假设，则采用低效率和噪声的乘法规范在逻辑上是一致的，接下来将讨论。

我们可以将模型 (10.1) 改写为：
$$
y_{i}=f\left(\mathbf{x}_{i}\right) \cdot \exp \left(\varepsilon_{i}\right)=f\left(\mathbf{x}_{i}\right) \cdot \exp \left(v_{i}-u_{i}\right)  \tag{10.23}
$$
如上所述，CRS 假设需要乘法误差结构。 此外，乘法误差规范可能有助于缓解不同尺度大小的异方差性（参见 Caudill 和 Ford 1993）。

我们对生产函数 f 和复合误差项保持与模型 (10.1) 中相同的假设。 对等式 （10.23）应用对数变换，我们得到
$$
\ln y_{i}=\ln f\left(\mathbf{x}_{i}\right)+\varepsilon_{i} .   \tag{10.24}
$$
我们仍然要求CNLS估计量估计的误差项的期望为0。因此，我们将公式（10.24）改写为
$$
\ln y_{i}=(\ln f\left(\mathbf{x}_{i}\right)-\mu)+(\varepsilon_{i}+\mu) 
=\ln g(\mathbf{x}_{i})+(\varepsilon_{i}+\mu)   \tag{10.25}
$$
其中$\mu$是预期的低效率，为了通过 CNLS 估计$\ln g$，我们可以将 QP 问题（10.3）改写为：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \phi, \varepsilon} \sum_{i=1}^{n}\left(\varepsilon_{i}^{C N L S}\right)^{2} \\
\text { subject to } \\
\ln y_{i}=\ln \left(\phi_{i}+1\right)+\varepsilon_{i}^{C N L S} \forall i \\
\phi_{i}+1=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i} \forall h, i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} \forall i
\end{array}   \tag{10.26}
$$
其中$ \phi_{i}+1 $是$ E\left(y_{i} \mid \mathbf{x}_{i}\right)=g(\mathbf{x}_{i}) $的CNLS估计器。这里加上1的值是为了确保计算算法不会试图取零的对数。第一个等式可以解释为对数转换的回归方程（使用自然对数函数$ln(.)$）。第二至第五个约束条件与(10.3)相似，但(10.3)中观察到的产出被替换为$ \phi_{i}+1 $。使用$ \phi_{i}$可以估计产出和投入之间的乘法关系，同时保证原始投入产出空间中生产可能性集的凸性。

在乘法模型下，可以应用CNLS估计器作为第10.2.3节描述的StoNED估计程序的第一步。接着使用第10.2.3节描述的StoNED估计程序的第二步估计非效率的均值$\mu$。在进行第三步之前，我们仍然使用最小外推方法，估计$g$的下限：
$$
\hat g_{min}^{CNLS}(\bold{x})=min_{\alpha,\bold{\beta}} \{\alpha+\bold{x'}|\alpha_i+\bold{x_i'}\geq\hat g^{CNLS}(\bold{x_i}) \forall i=1,...,n\}
$$
第三步，鉴于 $(\ln f\left(\mathbf{x}_{i}\right)-\mu) =\ln g(\mathbf{x}_{i})$ ，等价于$f\left(\mathbf{x}_{i}\right) / exp(\mu) = g(\mathbf{x}_{i})$，我们使用$ \hat{f}^{S t o N E D}\left(\mathbf{x}_{i}\right) =\hat{g}_{\min }^{C N L S}(\mathbf{x}) \cdot \exp (\hat{\mu}) $作为前沿$f$的估计量。

### 10.1.5 背景变量

以乘法模型为起点，我们引入背景变量（contextual variable）。用r维向量$ \mathbf{z}_{i} $表示背景变量的观测值，我们假设综合误误差项为$\varepsilon_{i}=v_{i}-u_{i}+ z'\delta $，从而得到以下半非参数化的偏对数线性方程：
$$
\ln y_{i}=\ln f\left(\mathbf{x}_{i}\right)+\boldsymbol{\delta}^{\prime} \mathbf{z}_{i}+v_{i}-u_{i}  \tag{10.27}
$$
其中，$\delta=(\delta_1,\ldots,\delta_r)'$是一个带估计的参数的向量，表示背景变量$ \mathbf{z}_{i} $对绩效的平均影响。还有其他的设置背景变量的方法，如将$ \mathbf{z} $变量影响随机非效率项$u_{i}$的概率分布。需要指出的是，这两种设置具有非常大的差异。我们这里主要涉及公式（10.27）中的设置方法。Johnson等(2011) [^4]认为公式（10.27）中的设置方法需要更少的假设条件。

背景变量$ \mathbf{z}_{i} $可以直接纳入StoNED问题中的目标函数，公式（10.27）可以使用与公式（10.3）相似的方程求解：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\delta}, \phi, \boldsymbol{\varepsilon}} \sum_{i=1}^{n}\left(\varepsilon_{i}^{C N L S}\right)^{2} \\
\text { subject to } \\
\ln y_{i}=\ln \left(\phi_{i}+1\right)+\boldsymbol{\delta}^{\prime} \mathbf{z}_{i}+\varepsilon_{i}^{C N L S} \forall i \\
\phi_{i}+1=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i} \forall h, i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} \forall i
\end{array} \tag{10.28}
$$
其中，$ \hat{\boldsymbol{\delta}}^{\text {StoNEZD }} $表示作为模型（10.27）的最优解得到的背景变量的系数。Johnson等(2011) [^4]详细研究了这个估计器的统计特性，显示了其无偏性、一致性和渐进效率。

### 10.1.6 凸非参数分位数回归

我们进一步定义凸非参数分位数回归模型，从而在数据内部分析投入的边际产出。

令$ Y \mid \mathbf{x} $ 是给定投入水平 $ \mathbf{x} $ 的随机产出$ Y $，其条件分布为$ F_{Y \mid \mathbf{x}} $，即：$ F_{Y \mid \mathbf{x}}(y) \triangleq \operatorname{Prob}\{Y \leqslant y \mid $  $ \mathbf{x}\} $。那么，在投入水平$ \mathbf{x} $的$\tau$-边界被定义为：
$$
\varphi_{\tau}(\mathbf{x}) \triangleq \inf \left\{y \in \mathbb{R}_{+} \mid F_{Y \mid \mathbf{x}}(y) \geqslant  \tau\right\} 
$$
如果$ F_{Y \mid \mathbf{x}} $是连续且严格增加的，我们有$ \varphi_{\tau}(\mathbf{x})=F_{Y \mid \mathbf{x}}^{-1}(1-\tau) $。$\tau$-边界的经济含义是很直观的：对于任何给定的投入水平  ，效率和非效率的百分比分别为$1-\tau$和$\tau$。

Wang等人（2014）[^11]通过类比线性分位数回归中关于目标函数的设定和凸非参数最小二乘中关于凸性和单调性假设的设定，定义了如下凸非参数分位数回归模型(CNQR)的求解方法：
$$
\begin{array}{l}
\min _{\alpha, \boldsymbol{\beta}, \varepsilon^{+}, \varepsilon^{-}} \tau \sum_{i=1}^{n} \varepsilon_{i}^{+}+(1-\tau) \sum_{i=1}^{n} \varepsilon_{i}^{-} \\
\text {subject to } \\
y_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\varepsilon_{i}^{+}-\varepsilon_{i}^{-} \forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i} \forall h, i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} \forall i \\
\varepsilon_{i}^{+} \geq 0 \forall i \\
\varepsilon_{i}^{-} \geq 0 \forall i
\end{array}              \tag{10.29}
$$
与CNLS相比，CNQR的一个主要的缺点是该问题的最优解不一定是唯一的，甚至对于观察到的数据点$ \left(\mathbf{x}_{i}, y_{i}\right), i=1, \ldots, n $也是如此。Kuosmanen等人（2015）[^12]建议使用以下的目标函数的二次方表述来求得唯一解：
$$
\begin{array}{l}
\min _{\alpha, \boldsymbol{\beta}, \varepsilon^{+}, \varepsilon^{-}} \tau \sum_{i=1}^{n} (\varepsilon_{i}^{+})^2+(1-\tau) \sum_{i=1}^{n} (\varepsilon_{i}^{-})^2 \\
\text {subject to } \\
y_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\varepsilon_{i}^{+}-\varepsilon_{i}^{-} \forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i} \forall h, i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0} \forall i \\
\varepsilon_{i}^{+} \geq 0 \forall i \\
\varepsilon_{i}^{-} \geq 0 \forall i
\end{array}             \tag{10.30}
$$
对异方差的稳健性被普遍认为是分位数回归的主要优势之一（例如Koenker，2005[^13]）。如果$\varepsilon_i$是同方差的，那么根据经验估计的分位数前沿的形状在不同的$\tau$下将是相似的。在这种情况下，估计一个单一的分位数指标或条件平均值就足以得到任何感兴趣的分位数指标。通过实证估计与不同绩效水平相关的多个分位数指标，可以有效地放松同方差假设，因为分位数前沿的形状可以在不同的$\tau$之间自由地变化。

## 10.2 多产出StoNED模型的设定

到目前为止，CNLS和StoNED框架一直是在单产出、多投入的情况下提出的。在本节中，我们将在Chambers等人（1996，1998）[^14] [^15]的定向距离函数（DDF）框架内描述CNLS估计器。CNLS 的表述从结构上满足了 DDF 的公理特性，对多投入和多产出进行建模，并明确说明了随机噪声，解决了 DEA 和参数化方法的关键限制。下面我们将简要介绍CNLS对DDF的估计。

考虑一种联合生产技术，将投入向量$ \mathbf{x}=\left(x_{1} \ldots x_{m}\right)^{\prime} \in \Re_{+}^{m} $转换为期望产出$ \mathbf{y}=\left(y_{1} \ldots y_{s}\right)^{\prime} \in \Re_{+}^{s} $ 。基准技术的特点是生产可能性集，定义为：
$$
T=\left\{(\mathbf{x}, \mathbf{y}) \in \mathfrak{R}_{+}^{m+s} \mid \mathbf{X} \text { 可以生产 }(\mathbf{y})\right\} .            \tag{10.31}
$$
本文考虑的基准技术的经典公理包括自由可处置性（即 $T=T+\Re_{+}^{m} \times \Re_{-}^{s}  $）和凸性（即）$ T=\operatorname{Conv}(T) $（见，例如，Färe & Primont, 1995）。

方向距离函数被用来表示多产出的生产函数。Chambers等人（1996，1998）[^14] [^15]介绍的方向性距离函数（DDF）定义为：
$$
\vec{D}\left(\mathbf{x}, \mathbf{y},  \mathbf{g}^{x}, \mathbf{g}^{y}\right)=\sup \left\{\theta \mid\left(\mathbf{x}-\theta \mathbf{g}^{x}, \mathbf{y}+\theta \mathbf{g}^{y}\right) \in T\right\}            \tag{10.32}
$$
其中$ \left(\mathbf{g}^{x}, \mathbf{g}^{y}\right) \in \Re_{+}^{m+s} $是一个方向向量。

DDF继承了生产可能性集的公理属性（见Chambers, Chung & Färe, 1998, Lemma 2.2）[^15]。任何DDF必须满足的两个基本属性包括转换属性（即：$ \vec{D}\left(\mathbf{x}-\lambda \mathbf{g}^{x}, \mathbf{y}+\lambda \mathbf{g}^{y}, \mathbf{g}^{x}, \mathbf{g}^{y}\right)=\vec{D}\left(\mathbf{x}, \mathbf{y},  \mathbf{g}^{x}, \mathbf{g}^{y}\right)-\lambda $ 和方向向量上的-1度齐次性（即$ \vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{b}, \lambda \mathbf{g}^{x}, \lambda \mathbf{g}^{y}\right)=   \left(1 / \lambda \right) \cdot \vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{g}^{x}, \mathbf{g}^{y}\right), \lambda>0 $ 。这些属性对DDF的内部一致性至关重要，必须在DDF的经验估计中强加这些属性。

Kuosmanen和Johnson（2017）[^12]引入了一个DDF的概率模型，其中所有的投入和产出$(\mathbf{x}, \mathbf{y})$都受到综合误差项$ \varepsilon_{i}$和位于生产前沿的投入产出$(\mathbf{x^*}, \mathbf{y^*})$的影响，其中综合误差项$ \varepsilon_{i}=u_{i}+v_{i} $包含非负非效率项$ u_{i} $和随机噪声项$ v_{i} $，即：
$$
\left(\mathbf{x}_{i}, \mathbf{y}_{i}\right)=\left(\mathbf{x}_{i}^{*}+\varepsilon_{i} {\mathbf{g}}_{i}^{x}, \mathbf{y}_{i}^{*}-\varepsilon_{i} {\mathbf{g}}_{i}^{y}\right) \forall i=1, \ldots, n,
$$
由于观察到的数据受到非效率和噪声的干扰，第i个DMU的DDF值是一个随机变量：
$$
\vec{D}_{T}\left(\mathbf{x}_{i}, \mathbf{y}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)=\varepsilon_{i} \quad \forall i            \tag{10.33}
$$
由于所有投入和产出都受到非效率和噪声的影响，因此，所有投入和产出都是内生变量，在进行估计时需要解决这一问题。我们可以利用DDF的转换属性，估计：
$$
\vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-\frac{y_{1i}}{g_1^y}=\varepsilon_{i}              \tag{10.34}
$$
即：
$$
\frac{y_{1i}}{g_1^y} = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-\varepsilon_{i}  \\
 = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-(v_i+u_{i})\\
  = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-v_{i}-u_i \\
    = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)+v_{i}-u_i\tag{10.35}
$$
其中，${\overrightarrow{\mathbf{x}}_i}=\mathbf{x_i}+\frac{y_{1i}}{g_1^y} \mathbf{g}^{x}$，${\overrightarrow{\mathbf{y}}_i}=\mathbf{y_i}-\frac{y_{1i}}{g_1^y} \mathbf{g}^{y}$。可以证明，此时，$\operatorname{Cov}\left({\overrightarrow{\mathbf{x}}_i}, \varepsilon_{i}\right)=0$，$\operatorname{Cov}\left({\overrightarrow{\mathbf{y}}_i}, \varepsilon_{i}\right)=0$。

类比单产出的StoNED问题，我们可以在第一步中使用二次规划问题计算方向距离函数的超平面。需要注意的是$E(\varepsilon_{i})=E(u_i+v_i)=\mu \neq0$，所以，我们需要先估计：
$$
\frac{y_{1i}}{g_1^y}=(\vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-\mu)-(\varepsilon_{i}-\mu)  \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right) + (v_{i}-u_i +\mu )\quad\forall i       \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right) + (\varepsilon_i +\mu )\quad\forall i   \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right) + \varepsilon_i ^{CNLS} \quad\forall i
\tag{10.36}
$$
可以发现上式与单产出的StoNED模型（10.13）一致，因此，可以按照同样的方法计算超平面、非期望非效率和期望非效率。

$\vec{D}_{T2}$的估计值$\hat{\vec{D}}_{T2}$的计算方法如下：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma},  \boldsymbol{\varepsilon^{CNLS}}, } \sum_{i=1}^{n} {(\varepsilon_{i}^{CNLS})}^2 \\
\text {s.t. } \\
\frac{y_{1i}}{g_1^y}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i}+\varepsilon_{i}^{CNLS}&\forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}^{\prime}{ }_{h} \overrightarrow{\mathbf{y}}_{i}&\forall i, h \\
\boldsymbol{\beta}_{i}^{\prime}  \mathbf{g}^{x}+\boldsymbol{\gamma}_{i}^{\prime} \mathbf{g}^{y}=1 &\forall i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0}, \boldsymbol{\gamma}_{i} \geq \mathbf{0}  &\forall i \\
\end{array}          \tag{10.37}
$$


第二步是像单产出的模型中一样，使用Fan等人(1996)或Hall和Simar(2002)的研究来估计非效率的期望的估计值$\hat\mu$。

第三步是从$\vec{D}_{T2}$的估计值$\hat{\vec{D}}_{T2}$中减去$\mu$的估计值$\hat\mu$。但首先，请注意，CNLS 估计器在观察点$\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}\right),\forall i=1,\ldots,n$上是唯一的，但在未观察的$\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}\right) $上不是。为了解决非唯一性问题，Kuosmanen 和 Kortelainen (2012)[^5] 求助于最小外推原则，并提出估计 CNLS 估计量拟合值的最小包络。同样，我们对 $\hat{\vec{D}}_{T2}^*$ 的最小外推估计$\hat{\vec{D}}_{T2,min}^*$由下式给出：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma}, \delta, \boldsymbol{\varepsilon^{CNLS}}, } \alpha+\boldsymbol{\beta}^{\prime} \mathbf{x}-\boldsymbol{\gamma}^{\prime} \mathbf{y} \\
\text {s.t. } \\
\alpha+\boldsymbol{\beta}^{\prime} \hat{\mathbf{x}}-\boldsymbol{\gamma}^{\prime} \hat{\mathbf{y}} \geq 0\\
\boldsymbol{\beta}^{\prime}  \mathbf{g}^{x}+\boldsymbol{\gamma}^{\prime} \mathbf{g}^{y}=1  \\
\boldsymbol{\beta} \geq \mathbf{0}, \boldsymbol{\gamma}\geq \mathbf{0}\\
\end{array}          \tag{10.38}
$$
其中，$\hat{\mathbf{x}_i}=\mathbf{x_i}-(\hat{\mu}+\hat\varepsilon_{i}^{CNLS}) \mathbf{g}^{x}$，$\hat{\mathbf{y_i}}=\mathbf{y_i}+(\hat{\mu}+\hat\varepsilon_{i}^{CNLS}) \mathbf{g}^{y}$。

第四步是使用$ \hat{\sigma}_{u} $ 和$ \hat{\sigma}_{v} $计算条件非效率 $ E\left(u_{i} \mid \varepsilon_{i}\right) $和效率$TE$：
$$
E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)=\frac{\hat{\sigma}_{u} \hat{\sigma}_{v}}{\sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\left[\frac{\phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}{1-\Phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}-\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right]   \tag{10.39}
$$
其中$ \phi $是标准正态分布$ N(0,1)$ 的密度函数，$\Phi$是相应的累积分布函数，且
$$
\hat{\varepsilon}_{i}=\hat{\varepsilon}_{i}^{C N L S}-\hat{\sigma}_{u} \sqrt{2 / \pi}
$$
得到非效率的条件期望值$E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)$后，可以使用产出的真实值与产出的条件前沿值之间的比值得到条件效率，在这里，$\frac{y_{1i}}{g_1^y}$为产出的真实值，产出的条件前沿值为$\frac{y_{1i}}{g_1^y}+E\left(u_{i} \mid \hat{\varepsilon}_{i}\right) $。因此，效率可以被计算为：
$$
TE=\frac{\frac{y_{1i}}{g_1^y}}{\frac{y_{1i}}{g_1^y}+E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)} \tag{10.40}
$$


## 10.3 有非期望产出的StoNEd模型的设定

考虑一种联合生产技术，将投入向量$ \mathbf{x}=\left(x_{1} \ldots x_{m}\right)^{\prime} \in \Re_{+}^{m} $转换为期望产出$ \mathbf{y}=\left(y_{1} \ldots y_{s}\right)^{\prime} \in \Re_{+}^{s} $ 和非期望产出 $ \mathbf{b}=\left(b_{1} \ldots b_{q}\right)^{\prime} \in \Re_{+}^{q} $。基准技术的特点是生产可能性集，定义为：
$$
T=\left\{(\mathbf{x}, \mathbf{y}, \mathbf{b}) \in \mathfrak{R}_{+}^{m+s+q} \mid \mathbf{X} \text { can produce }(\mathbf{y}, \mathbf{b})\right\} .           \tag{10.41}
$$
本文考虑的基准技术的经典公理包括自由可处置性（即 $T=T+\Re_{+}^{m} \times \Re_{-}^{s} \times \Re_{+}^{q} $）和凸性（即）$ T=\operatorname{Conv}(T) $（见，例如，Färe & Primont, 1995）[^16]。另外，假设非期望产出$\mathbf{b}$和投入$\mathbf{x}$是单调递增的，且期望产出是单调递减的。对非期望产出的单调性显然与物质平衡原则不一致：对于给定的物质投入，非期望产出不能无限增加（例如，Rødseth，2016，2017）。然而，我们这里假设单调性可以保证非期望产出的影子价格是非负的。另外，也可以不假设非期望产出的单调性，从而施加非期望产出的弱可处置性假设。

方向距离函数被用来表示多产出的生产函数。Chambers, Chung和Färe (1996), 1998)[^14] [^15]介绍的方向性距离函数（DDF）定义为：
$$
\vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{b}, \mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right)=\sup \left\{\theta \mid\left(\mathbf{x}-\theta \mathbf{g}^{x}, \mathbf{y}+\theta \mathbf{g}^{y}, \mathbf{b}-\theta \mathbf{g}^{b},\right) \in T\right\},           \tag{10.42}
$$
其中$ \left(\mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right) \in \Re_{+}^{m+s+q} $是一个方向向量。

DDF继承了生产可能性集的公理属性（见Chambers, Chung & Färe, 1998, Lemma 2.2） [^15]。任何DDF必须满足的两个基本属性包括转换属性（即：$ \vec{D}\left(\mathbf{x}-\lambda \mathbf{g}^{x}, \mathbf{y}+\lambda \mathbf{g}^{y}, \mathbf{b}-\lambda \mathbf{g}^{b}, \mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right)=\vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{b}, \mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right)-\lambda $ 和方向向量上的-1度齐次性（即$ \vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{b}, \lambda \mathbf{g}^{x}, \lambda \mathbf{g}^{y}, \lambda \mathbf{g}^{b}\right)=   \left(1 / \lambda \right) \cdot \vec{D}\left(\mathbf{x}, \mathbf{y}, \mathbf{b}, \mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right), \lambda>0 $ 。这些属性对DDF的内部一致性至关重要，必须在DDF的经验估计中强加这些属性。

Kuosmanen和Johnson（2017）[^4]引入了一个DDF的概率模型，其中所有的投入和产出$(\mathbf{x}, \mathbf{y}, \mathbf{b})$都受到综合误差项$ \varepsilon_{i}$和位于生产前沿的投入产出$(\mathbf{x^*}, \mathbf{y^*}, \mathbf{b^*})$的影响，其中综合误差项$ \varepsilon_{i}=u_{i}+v_{i} $包含非负非效率项$ u_{i} $和随机噪声项$ v_{i} $，即：
$$
\left(\mathbf{x}_{i}, \mathbf{y}_{i},\mathbf{b}_{i}\right)=\left(\mathbf{x}_{i}^{*}+\varepsilon_{i} {\mathbf{g}}_{i}^{x}, \mathbf{y}_{i}^{*}-\varepsilon_{i} {\mathbf{g}}_{i}^{y}, \mathbf{b}_{i}^{*}+\varepsilon_{i} {\mathbf{g}}_{i}^{b}\right) \forall i=1, \ldots, n \tag{10.43}
$$
由于观察到的数据受到非效率和噪声的干扰，第i个DMU的DDF值是一个随机变量：
$$
\vec{D}_{T}\left(\mathbf{x}_{i}, \mathbf{y}_{i}, \mathbf{b}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)=\varepsilon_{i} \quad \forall i            \tag{10.44}
$$

由于所有投入和产出都受到非效率和噪声的影响，因此，所有投入和产出都是内生变量，在进行估计时需要解决这一问题。我们可以利用DDF的转换属性，估计：
$$
\vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-\frac{y_{1i}}{g_1^y}=\varepsilon_{i}              \tag{10.45}
$$
即：
$$
\frac{y_{1i}}{g_1^y} = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-\varepsilon_{i}  \\
 = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-(v_i+u_{i})\\
  = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-v_{i}-u_i \\
    = \vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)+v_{i}-u_i\tag{10.46}
$$
其中，${\overrightarrow{\mathbf{x}}_i}=\mathbf{x_i}+\frac{y_{1i}}{g_1^y} \mathbf{g}^{x}$，${\overrightarrow{\mathbf{y}}_i}=\mathbf{y_i}-\frac{y_{1i}}{g_1^y} \mathbf{g}^{y}$，${\overrightarrow{\mathbf{b}}_i}=\mathbf{b_i}+\frac{b_{1i}}{g_1^b} \mathbf{g}^{b}$。可以证明，此时，$\operatorname{Cov}\left({\overrightarrow{\mathbf{x}}_i}, \varepsilon_{i}\right)=0$，$\operatorname{Cov}\left({\overrightarrow{\mathbf{y}}_i}, \varepsilon_{i}\right)=0$，$\operatorname{Cov}\left({\overrightarrow{\mathbf{b}}_i}, \varepsilon_{i}\right)=0$。

类比单产出的StoNED问题，我们可以在第一步中使用二次规划问题计算方向距离函数的超平面。需要注意的是$E(\varepsilon_{i})=E(u_i+v_i)=\mu \neq0$，所以，我们需要先估计：
$$
\frac{y_{1i}}{g_1^y}=(\vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, \overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-\mu)-(\varepsilon_{i}-\mu)  \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, \overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right) + (v_{i}-u_i +\mu )\quad\forall i       \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, \overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right) + (\varepsilon_i +\mu )\quad\forall i   \\
=\vec{D}_{T2}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, \overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right) + \varepsilon_i ^{CNLS} \quad\forall i
\tag{10.47}
$$
可以发现上式与单产出的StoNED模型（10.13）一致，因此，可以按照同样的方法计算超平面、非期望非效率和期望非效率。

$\vec{D}_{T2}$的估计值$\hat{\vec{D}}_{T2}$的计算方法如下：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma},  \boldsymbol{\varepsilon^{CNLS}}, } \sum_{i=1}^{n} {(\varepsilon_{i}^{CNLS})}^2 \\
\text {s.t. } \\
\frac{y_{1i}}{g_1^y}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i}+\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i}+\varepsilon_{i}^{CNLS}&\forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i} +\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i}\leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}^{\prime}{ }_{h} \overrightarrow{\mathbf{y}}_{i}+\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i} &\forall i, h \\
\alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \overrightarrow{\mathbf{x}}_{i}\geq0   &\forall i, h\\
\boldsymbol{\beta}_{i}^{\prime}  \mathbf{g}^{x}+\boldsymbol{\gamma}_{i}^{\prime} \mathbf{g}^{y}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{g}^{b}=1 &\forall i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0}, \boldsymbol{\gamma}_{i} \geq \mathbf{0} ,\boldsymbol{\delta}_{i}^{\prime}\geq\mathbf{0} &\forall i \\
\end{array}          \tag{10.48}
$$
相比于多产出StoNED模型对应公式(10.37)，公式(10.48)中多出了第三个约束条件，这个约束条件对应非期望产出和期望产出的弱可处置性。

第二步是像单产出的模型中一样，使用Fan等人(1996)[^16]或Hall和Simar(2002)[^9]的研究来估计非效率的期望的估计值$\hat\mu$。

第三步是从$\vec{D}_{T2}$的估计值$\hat{\vec{D}}_{T2}$中减去$\mu$的估计值$\hat\mu$。但首先，请注意，CNLS 估计器在观察点$\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}\right),,\forall i=1,\ldots,n$上是唯一的，但在未观察的$\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}\right) $上不是。为了解决非唯一性问题，Kuosmanen 和 Kortelainen (2012) [^5]求助于最小外推原则，并提出估计 CNLS 估计量拟合值的最小包络。同样，我们对 $\hat{\vec{D}}_{T2}^*$ 的最小外推估计$\hat{\vec{D}}_{T2,min}^*$由下式给出：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma}, \delta, \boldsymbol{\varepsilon^{CNLS}}, } \alpha+\boldsymbol{\beta}^{\prime} \mathbf{x}+\boldsymbol{\delta}^{\prime} \mathbf{b}-\boldsymbol{\gamma}^{\prime} \mathbf{y} \\
\text {s.t. } \\
\alpha+\boldsymbol{\beta}^{\prime} \hat{\mathbf{x}}+\boldsymbol{\delta}^{\prime} \hat{\mathbf{b}}-\boldsymbol{\gamma}^{\prime} \hat{\mathbf{y}} \geq 0\\
\alpha+\boldsymbol{\beta}^{\prime} \hat{\mathbf{x}}\geq 0\\
\boldsymbol{\beta}^{\prime}  \mathbf{g}^{x}+\boldsymbol{\delta}^{\prime} \mathbf{g}^{b}+\boldsymbol{\gamma}^{\prime} \mathbf{g}^{y}=1  \\
\boldsymbol{\beta} \geq \mathbf{0}, \boldsymbol{\gamma}\geq \mathbf{0}, \boldsymbol{\delta} \geq \mathbf{0} \\
\end{array}          \tag{10.49}
$$
其中，$\hat{\mathbf{x}_i}=\mathbf{x_i}-(\hat{\mu}+\hat\varepsilon_{i}^{CNLS}) \mathbf{g}^{x}$，$\hat{\mathbf{y_i}}=\mathbf{y_i}+(\hat{\mu}+\hat\varepsilon_{i}^{CNLS}) \mathbf{g}^{y}$，$\hat{\mathbf{b_i}}=\mathbf{b_i}-(\hat{\mu}+\hat\varepsilon_{i}^{CNLS}) \mathbf{g}^{b}$。

第四步是使用$ \hat{\sigma}_{u} $ 和$ \hat{\sigma}_{v} $计算条件非效率 $ E\left(u_{i} \mid \varepsilon_{i}\right) $和效率$TE$：
$$
E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)=\frac{\hat{\sigma}_{u} \hat{\sigma}_{v}}{\sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\left[\frac{\phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}{1-\Phi\left(\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right)}-\frac{\hat{\varepsilon}_{i} \hat{\sigma}_{u}}{\hat{\sigma}_{v} \sqrt{\hat{\sigma}_{u}^{2}+\hat{\sigma}_{v}^{2}}}\right]   \tag{10.50}
$$
其中$ \phi $是标准正态分布$ N(0,1)$ 的密度函数，$\Phi$是相应的累积分布函数，且
$$
\hat{\varepsilon}_{i}=\hat{\varepsilon}_{i}^{C N L S}-\hat{\sigma}_{u} \sqrt{2 / \pi}
$$
得到非效率的条件期望值$E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)$后，可以使用产出的真实值与产出的条件前沿值之间的比值得到条件效率，在这里，$\frac{y_{1i}}{g_1^y}$为产出的真实值，产出的条件前沿值为$\frac{y_{1i}}{g_1^y}+E\left(u_{i} \mid \hat{\varepsilon}_{i}\right) $。因此，效率可以被计算为：
$$
TE=\frac{\frac{y_{1i}}{g_1^y}}{\frac{y_{1i}}{g_1^y}+E\left(u_{i} \mid \hat{\varepsilon}_{i}\right)} \tag{10.51}
$$


### 10.3.1 有背景变量和非期望产出的StoNED模型

以第10.3节描述的非期望产出模型为起点，我们引入背景变量。用r维向量$ \mathbf{z}_{i} $表示背景变量的观测值，我们假设综合误误差项为$\varepsilon_{i}=v_{i}-u_{i}+ \eta^{\prime}\mathbf{z} $，从而得到以下方程：
$$
\vec{D}_{T}\left(\mathbf{x}_{i}, \mathbf{y}_{i}, \mathbf{b}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)=\varepsilon_{i}=v_{i}+u_{i}+ \eta^{\prime}\mathbf{z} \quad \forall i           \tag{10.52}
$$
由于所有投入和产出都受到非效率和噪声的影响，因此，所有投入和产出都是内生变量，在进行估计时需要解决这一问题。我们可以利用DDF的转换属性，估计：
$$
\vec{D}_{T}\left(\overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i},\overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y}\right)-\frac{y_{1i}}{g_1^y}=\varepsilon_{i}=v_{i}+u_{i}+ \eta^{\prime}\mathbf{z}          \tag{10.53}
$$
其中，${\overrightarrow{\mathbf{x}}_i}=\mathbf{x_i}+\frac{y_{1i}}{g_1^y} \mathbf{g}^{x}$，${\overrightarrow{\mathbf{y}}_i}=\mathbf{y_i}-\frac{y_{1i}}{g_1^y} \mathbf{g}^{y}$，${\overrightarrow{\mathbf{b}}_i}=\mathbf{b_i}+\frac{b_{1i}}{g_1^b} \mathbf{g}^{b}$。可以证明，此时，$\operatorname{Cov}\left({\overrightarrow{\mathbf{x}}_i}, \varepsilon_{i}\right)=0$，$\operatorname{Cov}\left({\overrightarrow{\mathbf{y}}_i}, \varepsilon_{i}\right)=0$，$\operatorname{Cov}\left({\overrightarrow{\mathbf{b}}_i}, \varepsilon_{i}\right)=0$。

我们可以将$ \eta^{\prime}\mathbf{z}$移到等式左边，并使用DDF的平移属性，可得：
$$
\vec{D}_{T}\left( \overrightarrow{\mathbf{x}}_{i}, \overrightarrow{\mathbf{y}}_{i}, \overrightarrow{\mathbf{b}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-\frac{y_{1i}}{g_1^y}- \eta^{\prime}\mathbf{z} \\=\vec{D}_{T}\left(\tilde{\overrightarrow{\mathbf{x}}}_{i}, \tilde{\overrightarrow{\mathbf{y}}}_{i}, \tilde{\overrightarrow{\mathbf{b}}}_{i}, {\mathbf{g}}_{i}^{x}, {\mathbf{g}}_{i}^{y},{\mathbf{g}}_{i}^{b}\right)-\frac{y_{1i}}{g_1^y}=v_{i}+u_{i} \quad \forall i     \tag{10.54}
$$
其中，$\tilde{\overrightarrow{\mathbf{x}}}_i=\overrightarrow{\mathbf{x}}_i-(\eta^{\prime}\mathbf{z}) \mathbf{g}^{x}$，$\tilde{\overrightarrow{\mathbf{y}}}_i=\overrightarrow{\mathbf{y}}_i+(\eta^{\prime}\mathbf{z}) \mathbf{g}^{y}$，$\tilde{\overrightarrow{\mathbf{b}}}_i=\overrightarrow{\mathbf{b}}_i-(\eta^{\prime}\mathbf{z}) \mathbf{g}^{b}$。

有背景变量和非期望产出的StoNED模型可以通过如下模型求解，
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma},  \boldsymbol{\varepsilon^{CNLS}}, } \sum_{i=1}^{n} {(\varepsilon_{i}^{CNLS})}^2 \\
\text {s.t. } \\
\frac{y_{1i}}{g_1^y}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i}+\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i}+\eta^{\prime}\mathbf{z}+\varepsilon_{i}^{CNLS}&\forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \overrightarrow{\mathbf{y}}_{i} +\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i}\leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \overrightarrow{\mathbf{x}}_{i}-\boldsymbol{\gamma}^{\prime}{ }_{h} \overrightarrow{\mathbf{y}}_{i}+\boldsymbol{\delta}_{i}^{\prime} \overrightarrow{\mathbf{b}}_{i} &\forall i, h \\
\alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \overrightarrow{\mathbf{x}}_{i}\geq0 \\
\boldsymbol{\beta}_{i}^{\prime}  \mathbf{g}^{x}+\boldsymbol{\gamma}_{i}^{\prime} \mathbf{g}^{y}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{g}^{b}=1 &\forall i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0}, \boldsymbol{\gamma}_{i} \geq \mathbf{0} ,\boldsymbol{\delta}_{i}^{\prime}\geq\mathbf{0} &\forall i \\
\end{array}          \tag{10.55}
$$
在得到$\varepsilon_{i}^{CNLS}$的估计值后，无条件非效率、条件非效率以及效率的值的估计都与第10.2节中的步骤一致。

### 10.3.2 有非期望产出的凸非参数分位数回归

在第10.2节的基础上，我们进一步定义凸非参数分位数回归模型，从而可以在数据内部分析投入的边际产出。我们通过求助于DDF的概率分位数特征，对含有低效率和噪音的随机数据进行建模：

$$
\begin{array}{l}
\vec{D}_{\tau}\left(\mathbf{x}_{i}, \mathbf{y}_{i}, \mathbf{b}_{i}, \mathbf{g}^{x}, \mathbf{g}^{y}, \mathbf{g}^{b}\right) \\
\quad=\sup \left\{\theta \mid \operatorname{Pr} \cdot\left[\left(\mathbf{x}_{i}-\theta \mathbf{g}^{x}, \mathbf{y}_{i}+\theta \mathbf{g}^{y}, \mathbf{b}_{i}-\theta \mathbf{g}^{b}\right) \in T\right] \geq 1-\tau\right\}
\end{array}     \tag{10.56}
$$
其中参数$ \tau \in[0,1] $。请注意，确定性DEA技术是通过设置$ \tau =1 $得到的极限特例。对分位数DDF的解释直接类似于标准线性分位数回归（Koenker & Bassett, 1978）[^17]。本公式可以看作是Banker、Datar和Kemerer（1991）[^18]以及Wang等人（2014）[^11]以前的分位数公式在一般多投入多产出设置中的延伸。

类比第10.1.6节的单产出的凸非参数分位数回归模型，可以定义了如下有非期望产出的凸非参数分位数回归模型(CNQR)的求解方法：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma}, \delta, \boldsymbol{\varepsilon}^{-}, \boldsymbol{\varepsilon}^{+}}(1-\tau) \sum_{i=1}^{n} \varepsilon_{i}^{-}+\tau \sum_{i=1}^{n} \varepsilon_{i}^{+} \\
\text {s.t. } \\
\boldsymbol{\gamma}_{i}^{\prime} \mathbf{y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{b}_{i}-\varepsilon_{i}^{-}+\varepsilon_{i}^{+} &\forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{b}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \mathbf{y}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}^{\prime}{ }_{h} \mathbf{b}_{i}-\boldsymbol{\gamma}^{\prime}{ }_{h} \mathbf{y}_{i} &\forall i, h \\
\boldsymbol{\beta}_{i}^{\prime}{ }_{i} \mathbf{g}^{x}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{g}^{b}+\boldsymbol{\gamma}_{i}^{\prime} \mathbf{g}^{y}=1 &\forall i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0}, \boldsymbol{\gamma}_{i} \geq \mathbf{0}, \boldsymbol{\delta}_{i} \geq \mathbf{0} &\forall i \\
\varepsilon_{i}^{-} \geq 0, \varepsilon_{i}^{+} \geq 0 &\forall i
\end{array}     \tag{10.57}
$$


与CNLS相比，CNQR的一个主要的缺点是该问题的最优解不一定是唯一的，甚至对于观察到的数据点$ \left(\mathbf{x}_{i}, y_{i}\right), i=1, \ldots, n $也是如此。Kuosmanen等人（2015）[^12]建议使用以下的目标函数的二次方表述来求得唯一解：
$$
\begin{array}{l}
\min _{\boldsymbol{\alpha}, \boldsymbol{\beta}, \boldsymbol{\gamma}, \boldsymbol{\delta}, \boldsymbol{\varepsilon}^{-}, \boldsymbol{\varepsilon}^{+}}(1-\tilde{\tau}) \sum_{i=1}^{n}\left(\varepsilon_{i}^{-}\right)^{2}+\tilde{\tau} \sum_{i=1}^{n}\left(\varepsilon_{i}^{+}\right)^{2}, \\
\text {subject to } \\
\boldsymbol{\gamma}_{i}^{\prime} \mathbf{y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{b}_{i}-\varepsilon_{i}^{-}+\varepsilon_{i}^{+} &\forall i \\
\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{b}_{i}-\boldsymbol{\gamma}_{i}^{\prime} \mathbf{y}_{i} \leq \alpha_{h}+\boldsymbol{\beta}_{h}^{\prime} \mathbf{x}_{i}+\boldsymbol{\delta}^{\prime}{ }_{h} \mathbf{b}_{i}-\boldsymbol{\gamma}^{\prime}{ }_{h} \mathbf{y}_{i} &\forall i, h \\
\boldsymbol{\beta}_{i}^{\prime}{ }_{i} \mathbf{g}^{x}+\boldsymbol{\delta}_{i}^{\prime} \mathbf{g}^{b}+\boldsymbol{\gamma}_{i}^{\prime} \mathbf{g}^{y}=1 &\forall i \\
\boldsymbol{\beta}_{i} \geq \mathbf{0}, \boldsymbol{\gamma}_{i} \geq \mathbf{0}, \boldsymbol{\delta}_{i} \geq \mathbf{0} &\forall i \\
\varepsilon_{i}^{-} \geq 0, \varepsilon_{i}^{+} \geq 0 &\forall i
\end{array}     \tag{10.58}
$$
同理，有非期望产出的凸非参数分位数回归可以添加背景变量，以分析背景变量的影响。

## 10.4 计算问题

对于模型（10.3），在样本中增加一个新的DMU会使未知参数的数量增加m+2（m为投入的维度），而Afriat不等式约束的数量增加2n（n为DMU的个数）。引入一个额外的输入变量会使未知参数的数量增加n，但对约束的数量没有影响。由于这些原因，标准的QP算法不足以处理有几百或几千个观测值的大样本。我们可以使用基于切割平面的算法来求解模型（10.3）。现在我们在下面解释该算法的各个步骤。

切割面算法是解决大规模优化问题的有效工具，这些问题的约束条件数量非常多。在我们继续之前，我们定义一些在大规模优化文献中常用的术语。在这里，主问题指的是具有n(n-1)个约束条件的主问题(10.3)，而减少的主问题指的是具有相同目标和变量的问题，但约束条件只是主问题的约束条件的子集。这些方法的主要思想是开始解决最初有几个约束条件的问题--最初的减少的主问题。然后，我们找到被违反的约束条件，并以延迟的方式迭代添加它们--在每次迭代中，我们解决一个减少的主问题（但约束条件逐渐多于最初的减少的主问题）。因此，这种方法在大规模优化文献中也被称为延迟约束生成（Bertsimas and Tsitsiklis 1997）。这种方法的成功在很大程度上取决于寻找被违反的约束条件的效率。

我们从n(n-1)个约束中的一小部分开始--一个初始的减少的主问题。通常情况下，只有n(n-1)个约束中的一小部分在主问题的最优解下是有效的，而且只用这些有效的约束来解决这个问题显然等同于解决完整问题。然而，这些有效的约束条件是事先不知道的。从一个 "接近 "有效约束集的约束集开始，一个关键的好处是，它可以大大减少在以后的迭代中增加的切割数量，并减少净计算负担。

有两种常用的方法来形成初始的n-1个约束：基于观测值之间的欧氏距离的跨越路径（SP）和观测值之间的最小生成树（MST）。此外，还有leeMoreEfficientAlgorithm2013提出的Sweet spot 方法。问题（10.3）的初始较少的主问题是：
$$
\begin{array}{l}
\min _{{\alpha}, \boldsymbol{\beta}, {\varepsilon}} \sum_{i=1}^{n}\left({\varepsilon_i^{CNLS}}\right)^{2} &\\
\text {s.t. } \\ 
{\alpha}_{i_{1}}+\boldsymbol{\beta}_{i_{1}}^{\prime}\boldsymbol{x}_{i_{2}} \geq {\alpha}_{i_{2}}+\boldsymbol{\beta}_{i_{2}}^{\prime}\boldsymbol{x}_{i_{2}}  & \\
{\alpha}_{i_{2}}+\boldsymbol{\beta}_{i_{2}}^{\prime}\boldsymbol{x}_{i_{3}} \geq {\alpha}_{i_{3}}+\boldsymbol{\beta}_{i_{3}}^{\prime}\boldsymbol{x}_{i_{3}}  & \\
{\alpha}_{i_{n-1}}+\boldsymbol{\beta}_{i_{n-1}}^{\prime}\boldsymbol{x}_{i_{n}} \geq {\alpha}_{i_{n}}+\boldsymbol{\beta}_{i_{n}}^{\prime}\boldsymbol{x}_{i_{n}}  & \\
 {y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{\boldsymbol{i}} +{\varepsilon_i^{CNLS}} &\forall i\\
   \boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i \\
 \varepsilon_{i}^{CNLS} \geq 0 & \forall i
\end{array}     \tag{10.59}
$$
对于由$\hat{\alpha}, \hat{\boldsymbol{\beta}}, \hat{\varepsilon}$给出的缩小的主问题（10.59）的解，我们需要检查这是否是主问题的可行解。如果它确实是可行的，那么显然它也是主问题的最优解。另一方面，如果它不可行，那么我们需要有效地找到一个违规的约束条件。这个寻找违规约束的问题也被称为分离问题，因为这相当于找到一个超平面，将$\hat{\alpha}, \hat{\boldsymbol{\beta}}, \hat{\varepsilon}$从可行集合中分离出来（Bertsimas和Tsitsiklis 1997）[^19]。因此，对于每一个i，第i个分离问题是找到最小的索引$m(i)$，满足
$$
m(i)=\arg \min _{1 \leq m \leq n}\left\{(\alpha_i+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{m})-(\alpha_m+\boldsymbol{\beta}_{m}^{\prime}\boldsymbol{x}_{m})<0\right\}    \tag{10.60}
$$
此外，当且仅当最小值小于一个给定的阈值时，违反的约束才会被添加到减少的主问题中。

添加的分离问题对应的约束条件为：
$$
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{m} \geq {\alpha}_{m}+\boldsymbol{\beta}_{m}^{\prime}\boldsymbol{x}_{m}  \quad \forall i
$$
我们用$ T_{k} $表示我们在第$k$次迭代时加入的被违反的约束的索引对。因此，在第k次迭代后，我们所解决的问题为：
$$
\begin{array}{l}
\min _{{\alpha}, \boldsymbol{\beta}, {\varepsilon}} \sum_{i=1}^{n}\left({\varepsilon_i^{CNLS}}\right)^{2} &\\
\text {s.t. } \\ 
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{j}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{0} \\
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{h}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{1} \\
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{h}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{k} \\
 {y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{\boldsymbol{i}} +{\varepsilon_i^{CNLS}} &\forall i\\
   \boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i \\
 \varepsilon_{i}^{CNLS} \geq 0 & \forall i
\end{array}         \tag{10.61}
$$
同样，在凸非参数分位数回归中，我们可以使用上述算法来求解大样本问题。需要做的是将目标函数对应，并改变相应的Afrait条件。问题（10.29）对应的减少的主问题为：
$$
\begin{array}{l}
\min _{\alpha, \boldsymbol{\beta}, \varepsilon^{+}, \varepsilon^{-}} \tau \sum_{i=1}^{n} \varepsilon_{i}^{+}+(1-\tau) \sum_{i=1}^{n} \varepsilon_{i}^{-} \\
\text {s.t. } \\ 
{\alpha}_{i_{1}}+\boldsymbol{\beta}_{i_{1}}^{\prime}\boldsymbol{x}_{i_{2}} \geq {\alpha}_{i_{2}}+\boldsymbol{\beta}_{i_{2}}^{\prime}\boldsymbol{x}_{i_{2}}  & \\
{\alpha}_{i_{2}}+\boldsymbol{\beta}_{i_{2}}^{\prime}\boldsymbol{x}_{i_{3}} \geq {\alpha}_{i_{3}}+\boldsymbol{\beta}_{i_{3}}^{\prime}\boldsymbol{x}_{i_{3}}  & \\
{\alpha}_{i_{n-1}}+\boldsymbol{\beta}_{i_{n-1}}^{\prime}\boldsymbol{x}_{i_{n}} \geq {\alpha}_{i_{n}}+\boldsymbol{\beta}_{i_{n}}^{\prime}\boldsymbol{x}_{i_{n}}  & \\
 {y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{\boldsymbol{i}} +\varepsilon_{i}^{+}-\varepsilon_{i}^{-}  &\forall i\\
   \boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i \\
 \varepsilon_{i}^{+} ,\varepsilon_{i}^{-}\geq 0 & \forall i
\end{array}       \tag{10.62}
$$
在第k次迭代后，我们所解决的问题为：
$$
\begin{array}{l}
\min _{\alpha, \boldsymbol{\beta}, \varepsilon^{+}, \varepsilon^{-}} \tau \sum_{i=1}^{n} \varepsilon_{i}^{+}+(1-\tau) \sum_{i=1}^{n} \varepsilon_{i}^{-} \\
\text {s.t. } \\ 
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{j}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{0} \\
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{h}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{1} \\
{\alpha}_{i}+\boldsymbol{\beta}_{i}^{\prime}\boldsymbol{x}_{h} \geq {\alpha}_{h}+\boldsymbol{\beta}_{h}^{\prime}\boldsymbol{x}_{h}  & \forall(i, h) \in T_{k} \\
 {y}_{i}=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{\boldsymbol{i}} +\varepsilon_{i}^{+}-\varepsilon_{i}^{-}  &\forall i\\
   \boldsymbol{\beta}_{i} \geq \mathbf{0} & \forall i \\
 \varepsilon_{i}^{+} ,\varepsilon_{i}^{-}\geq 0 & \forall i
\end{array}       \tag{10.63}
$$

### 10.5.3 返回值

与前面章节不同，CNLS 模型的 `optimize()` 不返回 DataFrame，而是将求解结果存储在模型对象的内部属性中。效率值需要通过 `StoNED` 类的方法获取。

以下按 **模型类型 × StoNED 方法** 对称列出。

#### CNLS 模型公共属性

`optimize()` 执行后，模型对象（`CNLSSDweak` / `CNLSDDFweak`）上可用的公共属性：

| 属性 | 类型 | 含义 |
|------|------|------|
| `optimization_status` | int | 求解状态（`1` 表示成功） |
| `problem_status` | int | 求解器问题状态 |

#### StoNED 工作流

CNLS 模型需要三步操作才能获得效率值：

```python
model.optimize(solver="...")                          # 1. 求解 CNLS 模型
rd = StoNED.StoNED(model)                              # 2. 创建 StoNED 对象
res = rd.get_technical_efficiency(method)              # 3. 获取效率
```

#### 残差分解方法（`method` 参数）

以下 3 种方法用于将 CNLS 残差 $\hat{\varepsilon}$ 分解为技术无效项 $\hat{u}$ 和统计噪声 $\hat{v}$：

| method 值 | 全称 | 返回类型 | 说明 |
|-----------|------|---------|------|
| `RED_QLE` | Quasi-Likelihood Estimation | — | 准似然估计（推荐），通过最大化修正对数似然函数估计 $\lambda$、$\sigma_u$、$\sigma_v$ |
| `RED_MOM` | Method of Moments | — | 矩方法，利用残差的二阶和三阶矩估计 $\sigma_u$、$\sigma_v$ |
| `RED_KDE` | Kernel Deconvolution Estimation | — | 核密度反卷积估计，使用 Richardson-Lucy 盲反卷积从残差中恢复 $\hat{u}$ |

> 以上 3 种方法不直接返回效率值，而是设置 `StoNED` 对象的内部属性（`sigma_u`, `sigma_v`, `mu`），供效率计算方法使用。

#### StoNED 效率获取方法

##### CNLSSDweak + `get_technical_efficiency(method)`（Shephard 距离函数）

| 返回项 | 类型 | 含义 |
|--------|------|------|
| 返回值 | numpy array | 技术效率 $TE = e^{-\hat{u}}$，每个 DMU 一个标量值 |

适用于 `CNLSSDweak`（`cet=CET_MULT`）。$\hat{u}$ 通过 JLLMS 估计量获得，$TE \in (0, 1]$，越接近 1 效率越高。

##### CNLSDDFweak + `get_technical_efficiency_ratio(method)` — gy 方向（期望产出扩张）

| 返回项 | 类型 | 含义 |
|--------|------|------|
| 返回值 | numpy array | 技术效率 $TE = \dfrac{y}{y + \hat{u}}$，其中 $y$ 为期望产出 |

当 `gy ≥ 1` 且 `gx = 0`、`gb = 0` 时使用。$\hat{u} > 0$ 表示产出扩张空间，$TE \in (0, 1]$。

##### CNLSDDFweak + `get_technical_efficiency_ratio(method)` — gx 方向（投入缩减）

| 返回项 | 类型 | 含义 |
|--------|------|------|
| 返回值 | numpy array | 技术效率 $TE = \dfrac{-x + \hat{\varepsilon}}{-x + \hat{\varepsilon} + \hat{u}}$，其中 $x$ 为投入 |

当 `gx ≥ 1` 且 `gy = 0`、`gb = 0` 时使用。$\hat{u} > 0$ 表示投入缩减空间，$TE \in (0, 1]$。

##### CNLSDDFweak + `get_technical_efficiency_ratio(method)` — gb 方向（非期望产出缩减）

| 返回项 | 类型 | 含义 |
|--------|------|------|
| 返回值 | numpy array | 技术效率 $TE = \dfrac{-b + \hat{\varepsilon}}{-b + \hat{\varepsilon} + \hat{u}}$，其中 $b$ 为非期望产出 |

当 `gb ≥ 1` 且 `gy = 0`、`gx = 0` 时使用。$\hat{u} > 0$ 表示非期望产出缩减空间，$TE \in (0, 1]$。

#### CNLSSDweak vs CNLSDDFweak 对比

| 特征 | CNLSSDweak | CNLSDDFweak |
|------|-----------|-------------|
| 距离函数类型 | Shephard 距离函数 | 方向距离函数 |
| cet 参数 | **必须** `CET_MULT`（乘法复合误差） | 无（内部 `CET_ADDI`，加法） |
| 求解器 | `"knitro"`（非线性） | `"mosek"`（线性） |
| StoNED 效率方法 | `get_technical_efficiency()` | `get_technical_efficiency_ratio()` |
| TE 公式 | $TE = e^{-\hat{u}}$ | $TE = \dfrac{w}{w + \hat{u}}$，$w$ 随方向变化（$y$/$x$/$b$） |
| 返回类型 | numpy array | numpy array |
| 适用方向 | 无方向概念 | gy（产出）、gx（投入）、gb（非期望产出） |

#### 4 种组合汇总

| 模型 | StoNED 方法 | 方向条件 | TE 公式 |
|------|------------|---------|---------|
| CNLSSDweak | `get_technical_efficiency()` | — | $e^{-\hat{u}}$ |
| CNLSDDFweak | `get_technical_efficiency_ratio()` | `gy ≥ 1` | $y / (y + \hat{u})$ |
| CNLSDDFweak | `get_technical_efficiency_ratio()` | `gx ≥ 1` | $(x + \hat{\varepsilon}) / (x + \hat{\varepsilon} + \hat{u})$ |
| CNLSDDFweak | `get_technical_efficiency_ratio()` | `gb ≥ 1` | $(b + \hat{\varepsilon}) / (b + \hat{\varepsilon} + \hat{u})$ |

> CNLSSDweak 的 3 种 `method`（QLE/MOM/KDE）均返回相同的 numpy array 结构，差异仅在 $\hat{u}$ 的估计方法。CNLSDDFweak 的 3 种方向返回相同的 numpy array 结构，差异仅在 $TE$ 公式中的 $w$（基准值）。

def CNLSDDFb_CRS():
    """
    构建基于CRS假设的CNLS方向性距离函数模型
    投入变量: K(资本), L(劳动), E(能源)
    产出变量: Y(期望产出), CO2(非期望产出)

### 10.6.1 基于 Shephard 距离函数的随机非参数前沿（CNLSSDweak）

前述§10.3至§10.5中的案例均采用方向距离函数（DDF）作为效率测度。本节介绍另一种基于**Shephard 距离函数**的随机非参数前沿模型——**CNLSSDweak**。该模型在 deabook 包中由 `CNLSSDFDDFweak.CNLSSDweak` 类实现。

#### （一）模型概念

Shephard 产出距离函数（Shephard, 1970）定义为：

$$
D_o(\mathbf{x},\mathbf{y},\mathbf{b})=\sup\left\{\theta>0 \mid (\mathbf{x},\mathbf{y}/\theta,\mathbf{b})\in T\right\}
$$

与 DDF 不同，Shephard 距离函数沿径向（即等比例）方向缩放产出，效率值 $\theta \geq 1$（$\theta=1$ 表示完全有效率）。DDF 则允许沿任意方向向量 $\mathbf{g}$ 非等比例地调整投入和产出，效率值可以为负（当 DMU 位于前沿之外时）。

在乘法误差结构下，Shephard 距离函数的 CNLS 估计可以表述为如下优化问题：

$$
\begin{array}{l}
\min \sum_{i=1}^{n}\left(\varepsilon_{i}^{CNLS}\right)^{2} \\

\text {s.t. } \\
\varepsilon_{i}^{CNLS}-\ln\left(\text{base}_{i}\right)=\ln\left(\hat{D}_{i}+1\right) \quad \forall i \\

\hat{D}_{i}+1=\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{i}-\boldsymbol{\gamma}_{i}^{\prime}\mathbf{y}_{i}+\boldsymbol{\kappa}_{i}^{\prime}\mathbf{b}_{i} \quad \forall i \\

\alpha_{i}+\boldsymbol{\beta}_{i}^{\prime}\mathbf{x}_{h}-\boldsymbol{\gamma}_{i}^{\prime}\mathbf{y}_{h}+\boldsymbol{\kappa}_{i}^{\prime}\mathbf{b}_{h} \leq \hat{D}_{h}+1 \quad \forall i, h \\

\boldsymbol{\beta}_{i}^{\prime}\mathbf{g}^{x}+\boldsymbol{\gamma}_{i}^{\prime}\mathbf{g}^{y}+\boldsymbol{\kappa}_{i}^{\prime}\mathbf{g}^{b}=1 \quad \forall i \\

\boldsymbol{\beta}_{i} \geq \mathbf{0},\;\boldsymbol{\gamma}_{i} \geq \mathbf{0},\;\boldsymbol{\kappa}_{i} \geq \mathbf{0} \quad \forall i
\end{array}
$$

其中，$\text{base}_i$ 为基准变量值（由方向向量决定，例如当 $g_b \neq 0$ 时取 $b_{1i}/g_1^b$），$\hat{D}_i+1$ 为 Shephard 距离函数值的 CNLS 估计量。注意，此处使用 $\ln(\hat{D}_i+1)$ 而非 $\ln(\hat{D}_i)$ 是为了避免取零的对数。

#### （二）CNLSSDweak 与 CNLSDDFweak 的区别

| 特征 | CNLSSDweak（Shephard 距离函数） | CNLSDDFweak（方向距离函数） |
|------|------|------|
| 距离函数类型 | Shephard 距离函数（径向、等比例缩放） | 方向距离函数 DDF（非径向、沿指定方向缩放） |
| 效率值含义 | $\theta \geq 1$，越接近 1 越有效率 | 可正可负，正值表示无效率程度 |
| 误差结构 | 乘法模型（对数变换，`CET_MULT`） | 可加（`CET_ADDI`）或乘法（`CET_MULT`） |
| 方向向量 | 仍使用 `gy`、`gx`、`gb` 指定归一化方向 | 同左 |
| 效率计算方法 | `StoNED.get_technical_efficiency(RED_QLE)` | `StoNED.get_technical_efficiency_ratio(RED_QLE)` |
| 适用场景 | 需要径向效率测度、与传统 DEA 径向模型对比 | 需要非径向效率测度、灵活指定调整方向 |

两者的核心差异在于：CNLSSDweak 基于 Shephard 距离函数的乘法模型框架，其效率计算采用指数形式（`np.exp(-inefficiency)`），效率值始终在 $(0,1]$ 区间内；而 CNLSDDFweak 基于 DDF 的框架，效率计算采用比值形式，效率值可能为负。

#### （三）代码示例

使用与前述§10.5相同的中国省份数据，以下代码展示 CNLSSDweak 的三种规模报酬设定：

```python
# 导入必要的模块
from deabook import CNLSSDFDDFweak, StoNED
from deabook.constant import FUN_PROD, OPT_LOCAL, RTS_VRS1, RTS_VRS2, RTS_CRS, CET_MULT, RED_QLE

# ===== CRS（规模报酬不变）=====
model_crs = CNLSSDFDDFweak.CNLSSDweak(
    data, sent="K L E=Y:CO2",       # 投入: K, L, E; 期望产出: Y; 非期望产出: CO2
    gy=[0], gx=[0,0,0], gb=[1],     # 仅沿非期望产出方向缩减
    cet=CET_MULT, rts=RTS_CRS, fun=FUN_PROD
)
model_crs.optimize(solver="knitro")   # 使用 Knitro 求解器
rd_crs = StoNED.StoNED(model_crs)
TE_crs = rd_crs.get_technical_efficiency(RED_QLE)  # 使用 QLE 方法估计效率

# ===== VRS 同方向（RTS_VRS1）=====
model_vrs1 = CNLSSDFDDFweak.CNLSSDweak(
    data, sent="K L E=Y:CO2",
    gy=[0], gx=[0,0,0], gb=[1],
    cet=CET_MULT, rts=RTS_VRS1, fun=FUN_PROD
)
model_vrs1.optimize(solver="knitro")
rd_vrs1 = StoNED.StoNED(model_vrs1)
TE_vrs1 = rd_vrs1.get_technical_efficiency(RED_QLE)

# ===== VRS 不同方向（RTS_VRS2）=====
model_vrs2 = CNLSSDFDDFweak.CNLSSDweak(
    data, sent="K L E=Y:CO2",
    gy=[0], gx=[0,0,0], gb=[1],
    cet=CET_MULT, rts=RTS_VRS2, fun=FUN_PROD
)
model_vrs2.optimize(solver="knitro")
rd_vrs2 = StoNED.StoNED(model_vrs2)
TE_vrs2 = rd_vrs2.get_technical_efficiency(RED_QLE)
```

> **注意**：CNLSSDweak 必须使用乘法复合误差项（`cet=CET_MULT`），否则将抛出错误。推荐使用 `knitro` 求解器以获得更好的收敛性。

#### （四）参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `data` | DataFrame | 包含投入、产出、非期望产出的 DMU 数据 |
| `sent` | str | 变量声明字符串，格式为 `"投入=期望产出:非期望产出"` |
| `z` | DataFrame/None | 背景变量（可选），默认为 None |
| `gy` | list | 产出方向向量，元素对应期望产出变量，0 表示不调整 |
| `gx` | list | 投入方向向量，元素对应投入变量，0 表示不调整 |
| `gb` | list | 非期望产出方向向量，元素对应非期望产出变量，0 表示不调整 |
| `cet` | str | 复合误差项类型，**必须为 `CET_MULT`**（乘法） |
| `fun` | str | 前沿类型：`FUN_PROD`（生产前沿）或 `FUN_COST`（成本前沿） |
| `rts` | str | 规模报酬假定：`RTS_CRS`（不变）、`RTS_VRS1`（可变-同方向）、`RTS_VRS2`（可变-不同方向） |

其中，`RTS_VRS1` 和 `RTS_VRS2` 的区别在于 VRS 约束的设定方式不同：`RTS_VRS1` 对应所有 DMU 共享相同的减排因子（reduction factor），而 `RTS_VRS2` 允许每个 DMU 具有不同的减排因子，提供了更大的灵活性。

## 10.7 本章总结

本章系统探讨了效率分析中非参数方法的前沿进展及其整合框架。传统方法中，数据包络分析（DEA）虽能避免参数设定却忽视随机噪声，随机前沿分析（SFA）虽可分解误差却依赖先验函数形式。凸非参数最小二乘法（CNLS）通过融合线性规划与二次规划技术，构建满足凸性、单调性公理的非参数前沿，既继承了DEA的模型灵活性，又引入SFA的统计推断框架。在此基础上，随机非参数数据包络（StoNED）通过误差项的概率建模与准似然估计，实现了生产技术的非参数估计与效率成分的统计分离，为效率分析提供了兼具严谨性与适应性的统一范式。

研究进一步扩展至多投入多产出系统，通过方向距离函数（DDF）建模环境效率，构建包含非期望产出的随机前沿模型，并引入分位数回归增强对异方差数据的适应性。针对大样本计算难题，切割平面算法通过动态生成有效约束显著提升求解效率。以中国省级碳排放效率为案例的实证表明，StoNED框架能有效识别环境效率的动态演变，其CRS与VRS模型的结果差异揭示了规模效应对效率评估的系统性影响。该方法不仅弥补了传统DEA对数据噪声敏感的缺陷，更通过引入方向向量与背景变量拓展了政策分析的维度，为复杂系统效率评估提供了新的方法论工具。
